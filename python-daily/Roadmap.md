## 🎯 Mission

This repository documents my **30-day journey** of learning Python for Cybersecurity while strengthening my Full-Stack Development skills.

The goal is to bridge the gap between core programming syntax and practical security operations, applying knowledge across critical security disciplines:

- **🔍 Threat Analysis:** Parsing unstructured logs and extracting meaningful information.
- **🌐 Security Tooling:** Network socket programming, port scanning, and banner grabbing.
- **🔒 Applied Cryptography:** Secure hashing, encryption, and secrets management.
- **⚡ Full-Stack Security:** Building React + Flask dashboards powered by Python security tools.
