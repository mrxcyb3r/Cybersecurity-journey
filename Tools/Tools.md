### 🛠️ Core Overview
Learned via NetworkChuck's 17-minute masterclass: [you need to learn NMAP right NOW!!](https://www.youtube.com/watch?v=4t4kBkMsDbQ). Nmap is an active reconnaissance utility used to discover hosts, map open ports, detect running services, and finger-print operating systems by analyzing raw packet responses.
